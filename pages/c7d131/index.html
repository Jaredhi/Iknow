masOS-MySql安装部署

# 1. 下载MySQL

  • Mysql官网下载 (opens new window)

  • 下载.dmg格式，默认安装到/usr/local/mysql。也可以下载tar.gz格式,自己选择安装位置。

  • 下载完成，默认一直下一步就行了，安装完后mysql会弹出一个框，告诉你安装成功以及设置root用户的初始密码。

# 2. 配置环境变量

  • 打开终端 cmd+空格

  • sudo 命令打开防止没有权限修改文件 sudo Vim /etc/profile

  • 然后配置 ,:wq退出（root用户且文件只读 ，:wq!强制退出）

    export MYSQL_HOME=/usr/local/mysql
export PATH=$PATH:$MYSQL_HOME/bin
# export PATH=@PATH:/usr/local/mysql/bin
    1
    2
    3

  • source ～/etc/profile让配置生效，$PATH查看是否有刚刚配置的路径

# 3. 启动MySql

  • 打开系统偏好设置

  • 启动MySql，绿点即已启动

  • 终端输入mysql -uroot -p
